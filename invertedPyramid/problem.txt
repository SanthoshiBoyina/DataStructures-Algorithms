Inverted Pyramid bookmark_borderPrint a hollow half-inverted pyramid pattern using '*'. See the example for more details.

Input Format
The first and only line of input contains a single integer N.

Output Format
For the given integer, print hollow half-inverted pyramid pattern.

Constraints
1 <= N <= 50

Input
5
