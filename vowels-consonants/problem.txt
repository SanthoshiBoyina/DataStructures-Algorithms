Vowels and Consonants bookmark_borderGiven a string, print the count of vowels and consonants in the string.

Input Format
Input contains a string S, consisting of lowercase and uppercase characters.

Output Format
Print the count of vowels and consonants in the given string, separated by space.

Constraints
1 <= len(S) <= 100

Example
Input
abxbbiAaspw

Output
4 7
